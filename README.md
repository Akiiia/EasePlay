# 视频播放程序开发文档

在开发过程中，我们了解到 WinForms 并没有自带的视频播放库，因此选择使用 `VideoLAN.LibVLCSharp` 来实现视频播放功能。

## 遇到的问题及解决方案

- **控件名称更改无自动更新**: 更改控件的 `Name` 属性时，`Form1.cs` 文件内没有自动更新相应内容。因此，需要手动调整代码中的函数名称。
